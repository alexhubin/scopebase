# ScopeBase Static UI Export

This folder contains a framework-free reference export of the current ScopeBase interface.

- `index.html` contains the landing page, authentication, workspace, project workflow, settings, and client portal screens.
- `styles.css` contains all design tokens, components, layouts, and responsive rules.
- The export has no JavaScript, backend calls, build step, or external package dependency.

Open `index.html` directly in a browser. Use the sticky screen index to jump between views. Each `.export-screen` block is self-contained enough to copy into another project while sharing the common stylesheet.

The files are a visual and structural reference. Forms, buttons, filters, and navigation are intentionally static.
